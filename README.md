## NOTE:

- Commit the files in the TESTING-BRANCH Only using the following format:

- Subject: Main topic of the changes
- Types of Changes:
    - "+" added files
    - "o" modified files
    - "-" removed files

```
<subject>

<type><reason>
```

- Example

```
Added The Landing Page

+ Created 3 new components
- Removed some dependencies
```
